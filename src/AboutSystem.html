Вклакда "Конструирование"

Вкладка конструирование предназначена для создания топологии парковки для дальнейшего моделирования из предоставленных функциональных блоков. В начале работы с вкладкой конструирования размер парковки задается автоматически и равен 4 блокам по вертикали и четырем блокам по горизонтали

Сверху располагается панель инструментов, на которой расположены функциональные кнопки: сохранение топологии парковки(1), задание размеров парковки(2), проверка топологии на корректность(3), загрузка ранее созданной топологии(4) и переход к моделированию(5).

Слева располагается панель с функциональными блоками: дорога, парковочное место, въезд, выезд, кассовый аппарат, информационное табло и газон. Функциональные блоки можно размещать на топологии, находящейся в центре рабочей области.

После создания топологии есть несколько вариантов действия: сохранить топологию парковки для дальнейшего изменения или загрузки во вкладке "Моделирование", переход во вкладку "Моделирование" с активной топологией, загрузка другой топологии для редактирования

Вклакда "Моделирование"

Вкладка "Моделирование" прездназначена для моделирования работы платной парковки с созданной во вкладке "Конструирование" структурой (топологией). Пример окна до начала моделирования представлен на рисунке ниже.

Сверху располагается панель инструментов, на которой расположены функциональные кнопки: загрузить существующую топологию(1), настроить параметры парковки(2), настроить параметры потока(3), запустить моделирование(4), поставить моделирование на паузу(5), остановить моделирование(6), открыть статистику моделирования(7).

После запуска начинается моделирование в соответствии с заданными параметрами. Кнопки паузы и остановки процесса моделирования доступны только после запуска моделирвания. Пример процесса моделирования показан ниже.
